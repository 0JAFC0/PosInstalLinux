# Automatizar-Pos-instalação-Do-ArchLinux

Arch-linux-Pos-Install
Shell Script de pós instalação do Arch Linux para utilização pessoal e também para aprendizagem de Shell.

Automatização de instalação:

- vscode
- brave
- okular
- figma
- typora
- postman
- gimp
- python3
- docker
- maven
- jdk8
- jdk11
- iriunwebcam
